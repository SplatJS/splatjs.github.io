<static> oscillate(current, period) → {number}

Oscillate between -1 and 1 given a value and a period. This is basically a simplification on using Math.sin().
Parameters:
Name Type Description
current number The current value of the number you want to oscillate.
period number The period, or how often the number oscillates. The return value will oscillate between -1 and 1, depending on how close current is to a multiple of period.
Source:
  • math.js, line 16
Returns:
A number between -1 and 1.
Type
number
Example
Splat.math.oscillate(0, 100); // returns 0
Splat.math.oscillate(100, 100); // returns 0-ish
Splat.math.oscillate(50, 100); // returns 1
Splat.math.oscillate(150, 100); // returns -1
Splat.math.oscillate(200, 100); // returns 0-ish
